Serverless Platform on Kubernetes
Build a FaaS (Function-as-a-Service) platform:

Core: Deploy Knative or OpenFaaS
Features:

Function deployment via API or CLI
Auto-scaling to zero for cost efficiency
Event-driven triggers (HTTP, Cron, message queues)
Multiple runtime support (Node.js, Python, Go)


Use Cases:

Webhook handlers
Scheduled data processing
API endpoints


Monitoring: Cold start metrics, execution duration, cost per function
UI: Dashboard for function management

Showcase Value: Trendy technology, shows innovation and understanding of serverless
